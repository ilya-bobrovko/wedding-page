# Свадьба — Арина и Илья

Одностраничное приглашение для GitHub Pages.

## Быстрый старт

1. Залейте репозиторий на GitHub.
2. **Settings → Pages → Build and deployment**: Source — **Deploy from a branch**, Branch — `main`, folder — `/ (root)`.
3. Сайт откроется по адресу `https://<username>.github.io/<repo>/` (если репозиторий не `username.github.io`).

## Редактирование текстов

Все тексты, время, ссылки и контакты — в файле **`content.json`**.

| Поле | Что менять |
|------|------------|
| `couple` | Имена |
| `hero` | Фраза и дата (строка для отображения) |
| `countdown.target` | Момент отсчёта (сбор гостей), формат ISO с часовым поясом |
| `venue` | Название, адрес, ссылка на карты, путь к фото |
| `schedule.items` | Программа дня |
| `wishes.items` | Пожелания |
| `organizer` | Контакт организатора (телефон и Telegram) |
| `footer` | Финальная фраза |

## Фон шапки

Файл **`assets/bg-header.png`** — водяной фон в верхней части страницы. Замените файл, сохранив имя, или поменяйте путь в `styles.css` (`--hero-bg-image`).

## Фото площадки

1. Положите файл в **`assets/venue.jpg`** (или укажите другой путь в `venue.image`).
2. Рекомендуемый размер: **1200×750 px** (или близко к 16:10), JPG или WebP, до ~300 КБ.
3. Если файла нет — показывается оливковая заглушка.

## Локальный просмотр

```bash
cd wedding-page
python3 -m http.server 8080
```

Откройте http://localhost:8080 (нужен локальный сервер из‑за загрузки `content.json`).
